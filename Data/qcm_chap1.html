QCM : Introduction                

Remarque : il faut valider chaque question

Question :
Quelles sont les propriétés incorrectes d'un SGBD ?
Réponses :
Non Cohérence de données
Indépendance logique
Non Redondance des Données
Indépendance physique

Question :
Quelles sont les objectifs d'une base de données?
Réponses :
Elimination de la redondance des données
Indépendance entre les programmes et les données
Intégration de données
Integration données et traitements

Question :
Quelles sont les trois nivaux de l'architecture d'un SGBD?
Réponses :
Conceptuel
Logique
Physiques
Traitement

Question :
Quelle est la définition de l'indépendance physique d'un SGBD?
Réponses :
La façon dont les données sont définies doit être indépendante des structures de stockage utilisées.
La façon dont les données sont définies doit être dépendante des structures de stockage utilisées
La façon dont les données sont définies doit être dépendante des structures de tables
La façon dont les données sont définies doit être différente des structures de stockage utilisées

Question :
Quelle est la définition de la cohérence de données d'un SGBD?
Réponses :
Les données sont soumises à un certain nombre de contraintes d'integrité 
Pas de redondance de données
Les données sont séparées aux traitement 
Les données sont indépendantes physiquement de l'espace de stockage
