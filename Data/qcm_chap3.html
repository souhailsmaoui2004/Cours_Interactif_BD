QCM : Le modèle relationnel            

Question :
Les contraintes d'integrité référentielle :
Réponses :
Vérifient la correspendance entre les valeurs d'une clé étrangère avec celle de la clé primaire correspondante
Vérifient l'unicité de la clé étragère
Vérifient l'unicité de la clé primaire
Vérifient que la valeur d'une clé étrangère est non nulle

Question :
Les contraintes d’intégrités  obligatoires correspondent aux
Réponses :
Contraintes d'intégrités sur les clés
Contraintes d'intégrités sur les entités
Contraintes d'intégrités référentielle
Contraintes d'intégrités sur les domaines

Question :
Contrainte d'intégrité sur les entités exige que :
Réponses :
La clé primaire ne doit pas avoir la valeur nulle
La clé étrangère doit être non nulle
La clé primaire doit être unique
La clé étrangère doit être unique

Question :
Une clé étrngère
Réponses :
Fait référence à une clé primaire
Doit être de valeur non nulle
Doit être formé uniquement d'un seul champs
Doit être unique

Question :
La clé d'une relation:
Réponses :
Toute table doit obligatoirement avoir une clé primaire 
C'est un attribut/groupe d’attributs qui détermine un tuple d’une manière unique dans la table
Est souvent soulignée 
Doit être toujours référencée au moins par une clé étrangère
